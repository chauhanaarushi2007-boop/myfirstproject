# **App Name**: CodeLingua

## Core Features:

- Language Tutorials: Comprehensive tutorials for various programming languages (HTML, Java, JavaScript, Python, C, C++, CSS, PHP, XML), based on aggregated content from sources like W3Schools and GeeksforGeeks.
- Coding Quizzes: Interactive quizzes for each language to test user knowledge.
- Important Questions: Curated list of frequently asked questions for each language to help with interviews and exams.
- Live Code Runner: Integrated live code runner for HTML, Java, JavaScript, Python, C, C++, CSS, and PHP, enabling users to execute code snippets directly in the browser.
- AI Chatbot 'MIA': An AI chatbot that serves as a tool to answer user queries related to programming languages and provide code explanations. The chatbot will be able to synthesize information from provided tutorials and external websites, using reasoning to determine the most appropriate responses and code examples.
- Language Documentation Aggregation: Multiple web pages with content aggregated from W3Schools, GeeksforGeeks, and other resource websites.
- Reference Books: Recommendations for the best books to learn more about each programming language.

## Style Guidelines:

- Primary color: Deep purple (#673AB7), conveying sophistication and modernity, reflecting the intellectual nature of coding.
- Background color: Light gray (#F5F5F5), offering a clean, distraction-free backdrop for focused learning.
- Accent color: Electric green (#76FF03), drawing users' attention to interactive elements.
- Body text: 'Inter', sans-serif.
- Headline text: 'Space Grotesk', sans-serif.
- Use clean, modern icons representing different programming languages and concepts.
- Employ a clean and responsive layout with clear navigation for easy access to tutorials, quizzes, and the chatbot.
- Use subtle transitions and animations to enhance user experience.
